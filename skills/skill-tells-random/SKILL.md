# Random Joke Skill

## When to use
Use this skill when the user asks to hear a joke, wants something funny,
or asks you to tell them a random joke.

## Inputs

```yaml
type: object
properties: {}
required: []
```

## How to respond
Deliver the joke naturally for spoken output. For setup/punchline jokes,
pause briefly between the setup and punchline. Keep it light and fun.
